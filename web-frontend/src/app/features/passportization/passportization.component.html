Паспортизация

Отчёты по ЛЭП, оборудованию и обходам, выгрузки для анализа и справочники марок — в одном месте. Здесь же ведутся марки проводов линий и марки оборудования; технические паспорта ЛЭП, опор и подстанций — со снимком данных и выгрузкой в PDF, Word и Excel.

У вас доступ только для просмотра: создание и изменение справочников и паспортов недоступны.

assessment Отчёты

Дефекты, сводка по линии, журнал обходов — JSON и CSV где доступно.

 badge Паспорта

Формирование технического паспорта по объекту (снимок из БД + разделы по СТП). Выгрузка: PDF и Word — по разделам; Excel для ЛЭП — по шаблону СТП (stp_passport_vl.xlsx), для опоры и ПС — листы по разделам.

precision_manufacturing Марки оборудования

Справочник типоразмеров для карточек оборудования: импорт, экспорт, ручное редактирование. Справочник типоразмеров для карточек оборудования (только просмотр).

cable Марки проводов ЛЭП

Справочник марок с расчётными параметрами (r/x/b, токи), импорт из Excel, вывод из эксплуатации. Номинальные марки проводников по напряжению (только просмотр).
